Paul's Dotfiles
================

###  Before you start...
Install a few things:

pip:
> sudo apt install pip
requests:
> pip install requests
rofi:
> sudo apt install rofi
terminator:
> sudo apt install terminator

### Run setup script
Make sure you run it with sudo!

> sudo ./setup
